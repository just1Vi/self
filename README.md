# days — локальное оффлайн‑приложение для ежедневного самоанализа

Минималистичный однофайловый проект (`index.html`) без бэкенда. Все записи хранятся локально как `.txt` файлы формата `YYYY-MM-DD.txt` в подпапке `days` выбранной директории. Работает оффлайн, без сетевых запросов.

## Запуск локально

1. Откройте терминал в каталоге проекта и запустите простой веб‑сервер:
   
   ```bash
   python -m http.server 8000
   ```

2. Откройте в браузере `http://localhost:8000` и загрузите `index.html`.

3. Нажмите «Выбрать папку days» и укажите любую локальную папку. Приложение создаст подпапку `days` внутри неё и будет сохранять туда файлы вида `YYYY-MM-DD.txt`.

Если ваш браузер не поддерживает File System Access API — появится режим «Fallback»: записи будут сохраняться во временный `localStorage`, а для выгрузки доступна кнопка «Скачать .txt» (для конкретной записи) и «Экспортировать всё» (ZIP со всеми `.txt`).

## Формат файла `.txt` (строго)

Каждый файл должен содержать поля в следующем порядке:

```
Дата: 2025-08-10
Время подъёма: 07:15
Как прошел поднятие: Легко — зарядка и свет в комнате помогли
Утренний ритуал: Молитва — выполнено; мотивация — выполнена; разбор — пропущен (не хватило времени)
Работа: Сел в 09:00, работалось продуктивно, сделал: A, B, C
Работоспособность: 7/10 — успел основное, не успел отчёт
Свободное время: Прогулка 30 минут, чтение 20 минут
Вечер: Подготовка ко сну выполнена: лёгкие растяжки, медитация
Время отхода ко сну: 23:30
Ключевые ошибки дня: Отвлечение на соцсети; не распланировал перерывы
Решения на завтра: Выключать уведомления в 10:00; сделать тайм-блоки
savedAt: 2025-08-10T23:40:12+02:00
appVersion: 1.0.0
```

Приложение формирует этот шаблон автоматически и поддерживает редактирование/пересохранение.

## Возможности

- Выбор локальной папки и сохранение в `days/` как `.txt` (File System Access API)
- Fallback, если FS API недоступен: `localStorage` + «Скачать .txt» и «Экспортировать всё (ZIP)»
- Список записей по датам и открытие/редактирование прошлых
- Автосохранение каждые ~20 с или при изменениях, Ctrl/Cmd+S
- Разрешение конфликтов: «Перезаписать / Создать копию (-v2) / Объединить»
- Пометка записи как «важная» (локально)
- Кнопка «Сохранить и закрыть»
- Подсчёт «стрика» — последовательных дней с записью
- Минималистичный UI с light/dark, анимацией статуса сохранения, доступностью и клавиатурной навигацией

## Поддерживаемые браузеры

- Chrome / Edge / Opera (FS Access API поддерживается)
- Safari, Firefox: работают в режиме Fallback (скачивание `.txt` и экспорт ZIP).

Если нужен полноценный доступ к файловой системе на всех платформах — используйте Electron‑сборку (см. ниже) или Chromium‑браузер.

## Восстановление из экспорта

1. Распакуйте ZIP, внутри будут файлы в папке `days/` вида `YYYY-MM-DD.txt`.
2. В приложении нажмите «Выбрать папку days» и укажите директорию, где лежит распакованная подпапка `days`.
3. Список записей обновится, вы сможете открывать и редактировать файлы.

## Горячие клавиши

- Ctrl/Cmd + S — сохранить текущую запись.

## Опционально: Electron

Создайте каталог `electron/` рядом с `index.html` со следующими файлами:

`electron/package.json`
```json
{
  "name": "days-app",
  "version": "1.0.0",
  "main": "main.js",
  "private": true,
  "scripts": {
    "start": "electron ."
  },
  "devDependencies": {
    "electron": "^31.0.0"
  }
}
```

`electron/main.js`
```js
const { app, BrowserWindow } = require('electron');
const path = require('path');

function createWindow () {
  const win = new BrowserWindow({
    width: 1100,
    height: 800,
    webPreferences: { nodeIntegration: false, contextIsolation: true }
  });
  win.loadFile(path.join(__dirname, '..', 'index.html'));
}

app.whenReady().then(() => {
  createWindow();
  app.on('activate', () => {
    if (BrowserWindow.getAllWindows().length === 0) createWindow();
  });
});

app.on('window-all-closed', () => {
  if (process.platform !== 'darwin') app.quit();
});
```

Установка и запуск:

```bash
cd electron
npm i
npm start
```

В Electron будут работать все функции, включая доступ к локальной файловой системе через встроенный Chromium.
